# `<ug-button>`

## Installation

```sh
npm install ug-button
```

## Example Usage

```html
<ug-button label="Click me!"></ug-button>

## API

### Properties/Attributes

| Name    | Type     | Default | Description                                      |
| ------- | -------- | ------- | ------------------------------------------------ |
| `label` | `string` | `''`    | Label to display for the button and `aria-label` |
