# Footer

The footer component is used to display the footer of a page. It contains links to various resources and information about the University of Guelph.

## Usage

### Basic

<Canvas of={FooterStories.Basic} sourceState='shown' />

### With Links

The footer also allows for a collection of links to be displayed above the main footer content.

<Canvas of={FooterStories.WithLinks} sourceState='shown' />

### Ridgetown Variant

Used for University of Guelph Ridgetown Campus sites, this variant of the footer includes a different logo and has content that is more relevant to University of Guelph Ridgetown.
